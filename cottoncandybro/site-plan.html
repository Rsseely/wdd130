Typography

Heading Font: Pacifico, Sacramento, Arial

Paragraph Font: Fresca, Roboto, Arial

Normal paragraph example

I chose the Pacifico font for the heading because I wanted to link the text from the logo to the rest of the website. I added Sacramento because it closely resembels the previous fonts. If the browser can't read the first two fonts I have Arial as my last resort because it can be read by every browser. For my paragraph fonts I chose Fresca as my maine font because it is a little fun but still very readable. A Website about cotton candy needs to be fun! I have Roboto and Arial as backup fonts.
